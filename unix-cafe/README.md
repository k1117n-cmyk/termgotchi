# UNIX Cafe Article Bundle

このディレクトリは、UNIX Cafe 掲載用の記事素材をまとめたものです。

## 収録ファイル

- `unix-cafe-article-ja.md`
  通常版の日本語記事原稿です。編集しながら仕上げたい場合はこちらを基準にします。

- `unix-cafe-article-ja-wp.md`
  WordPress にそのまま貼り付けやすい形に整えた版です。

- `unix-cafe-publish-kit-ja.md`
  推奨タイトル、リード文、短縮版抜粋をまとめた公開用キットです。

- `unix-cafe-title-ideas-ja.md`
  記事タイトル案の一覧です。

- `unix-cafe-eyecatch-copy-ja.md`
  アイキャッチ画像やバナーに使える文言案です。

## おすすめの使い方

1. まず `unix-cafe-publish-kit-ja.md` でタイトルとリード文を確認する
2. 本文は `unix-cafe-article-ja-wp.md` を WordPress に貼り付ける
3. 必要に応じて `unix-cafe-title-ideas-ja.md` から別タイトルを選ぶ
4. 画像まわりは `unix-cafe-eyecatch-copy-ja.md` を使う

## 最短導線

すぐに使うなら、次の 2 ファイルを開けば足ります。

- `unix-cafe-publish-kit-ja.md`
- `unix-cafe-article-ja-wp.md`
